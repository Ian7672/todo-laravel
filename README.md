# Todo Web Application

A modern, elegant, and luxurious todo web application built with Laravel. This application features a sleek glassmorphism design with smooth animations and a polished user interface.

## Features

- **Modern UI Design**: Implements glassmorphism effects, subtle gradients, and refined corner radii
- **Responsive Layout**: Works seamlessly across desktop and mobile devices
- **User Authentication**: Secure signup and login functionality
- **Task Management**: Create, read, update, and delete tasks with status tracking
- **Advanced Filtering**: Sort and filter tasks based on various criteria
- **Interactive Modals**: Smooth modal dialogs for task operations
- **Dark Mode Support**: Automatic dark mode based on system preferences

## Design Highlights

- **Glassmorphism Effects**: Elegant frosted glass UI elements
- **Subtle Animations**: Smooth transitions and hover effects
- **Soft Corners**: Moderately rounded corners for a refined look
- **Reduced Gradients**: Minimal use of gradients for cleaner aesthetics
- **Luxury Aesthetics**: Premium visual design with attention to detail

## Tech Stack

- **Backend**: Laravel PHP Framework
- **Frontend**: HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with modern techniques
- **Database**: MySQL
- **UI Components**: Bootstrap with custom enhancements

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ian7672/todo-webcopy.git
   ```

2. Navigate to the project directory:
   ```bash
   cd todo-webcopy
   ```

3. Install PHP dependencies:
   ```bash
   composer install
   ```

4. Install Node.js dependencies:
   ```bash
   npm install
   ```

5. Copy the environment file and generate application key:
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

6. Configure your database settings in the `.env` file

7. Run database migrations:
   ```bash
   php artisan migrate
   ```

8. Start the development server:
   ```bash
   php artisan serve
   ```
